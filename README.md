
# ps2023-tools
该项目是位ps2023 提供自定义插件功能

## 开发
请使用pnpm
```sh
pnpm install
```

## 运行分支
开发环境:
```sh
pnpm dev
```
测试环境:
```sh
pnpm dev:test
```
生产环境:
```sh
pnpm dev:prod
```

## 打包编译
开发环境:
```sh
pnpm build
```
测试环境:
```sh
pnpm build:test
```
生产环境:
```sh
pnpm build:prod
```
